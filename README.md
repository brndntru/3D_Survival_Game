# 3D Survival Game

Title: The Whiteout
Team: Hank Cao, Brandon Trieu
Platform(s): PC
Engine/Tools: Unity, C#, Github

## Overview

This project is a narrative-driven, semi-horror **3D survival game** built in **Unity**.  
The player must explore the tundra, maintain warmth, and uncover clues about the last remnants of humanity.

## Current Version: 1.0.0

### Core Features:
- **Controls**
  - first-person camera
  - WASD movement, sprinting, jumping
- **Survival Systems**
  - Health bar
  - Freeze bar
- **Mechanics**
  - Item-pickup

## Setup Instructions

Required platform: PC

Two options: Download folder with executable or clone git repository.

If download executable, open the game application and begin playing.

If cloning git repository:
  1. Open Unity Hub
  2. Click Add Project -> Add project from disk -> Select cloned folder
  3. Open the project (project built with Unity 2022.3.50f1)
  4. Play in edtor
